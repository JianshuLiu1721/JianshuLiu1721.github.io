Position Description

This position focuses on building efficient, intelligent, and reliable infrastructures for modern containerized and serverless platforms.

Qualifications

  • A Bachelor's/Master's degree (or equivalent) in Computer Science, Electrical/Computer Engineering, or a related field.
  • Background in cloud computing, distributed systems, or performance engineering.
  • Proficiency in programming languages such as Python, Java, or C++ and Linux environment.
  • Experience with container orchestration platforms (e.g., Kubernetes, Docker)
